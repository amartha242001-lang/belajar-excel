📙 Level 3: Olah Data

Kelola, bersihkan, dan atur data dengan fitur-fitur powerful Excel.

Data

1. Sorting (Pengurutan)

Sorting mengurutkan data berdasarkan satu atau beberapa kolom. Bisa dari A-Z, Z-A, terkecil-terbesar, atau custom.

CaraLangkah
Sort CepatKlik header kolom → Data → Sort A to Z / Z to A
Sort Multi-LevelData → Sort → Add Level (urutkan berdasarkan beberapa kolom)
Sort CustomData → Sort → Order → Custom List (misal: Senin, Selasa, Rabu...)

💡 Tips:

  • Pastikan data punya header agar Excel tidak ikut mengurutkan baris judul
  • Gunakan Ctrl+Shift+L untuk aktifkan filter, lalu sort dari dropdown
Data

2. Filtering (Penyaringan)

Filter menyembunyikan baris yang tidak sesuai kriteria tanpa menghapusnya. Data asli tetap utuh.

Cara Aktifkan:
Pilih data → Data → Filter (atau Ctrl+Shift+L)
Jenis FilterFungsi
Text FilterContains, Begins With, Ends With
Number FilterGreater Than, Less Than, Between
Date FilterThis Month, Last Quarter, Custom
Color FilterFilter berdasarkan warna sel atau font
Data

3. Data Validation

Data Validation membatasi input yang bisa dimasukkan ke sel. Berguna untuk membuat dropdown list atau mencegah input salah.

Cara:
Pilih sel → Data → Data Validation → Settings
Tipe ValidasiContoh Penggunaan
List (Dropdown)Pilihan: "Jakarta, Bandung, Surabaya"
Whole NumberHanya angka bulat 1-100
DateHanya tanggal setelah 01/01/2026
Text LengthMaksimal 50 karakter
Custom Formula=LEN(A1)<=10
Data

4. Remove Duplicates

Menghapus baris duplikat dari data. Excel akan menyimpan satu baris unik dan menghapus sisanya.

Cara:
Pilih data → Data → Remove Duplicates → Pilih kolom → OK

💡 Tips:

  • Backup data sebelum remove duplicates (tidak bisa di-undo setelah save)
  • Pilih kolom spesifik jika hanya ingin cek duplikat berdasarkan kolom tertentu
  • Alternatif: gunakan Conditional Formatting → Highlight Duplicates untuk melihat dulu sebelum hapus
Data

5. Text to Columns

Memisahkan isi satu kolom menjadi beberapa kolom berdasarkan delimiter (pemisah) tertentu.

Cara:
Pilih kolom → Data → Text to Columns → Delimited → Pilih separator (Comma, Space, dll)

Contoh:

"Andi,Jakarta,Marketing" → Kolom A: Andi | Kolom B: Jakarta | Kolom C: Marketing
