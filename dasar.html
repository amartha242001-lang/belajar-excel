📗 Level 1: Dasar Excel

Fondasi penting sebelum melangkah ke rumus dan analisis data.

Dasar

1. Mengenal Interface Excel

Komponen Utama:

  • Ribbon — Menu utama berisi tab (Home, Insert, Page Layout, Formulas, Data, Review, View)
  • Formula Bar — Tempat menulis/mengedit rumus, menampilkan isi sel aktif
  • Name Box — Menampilkan alamat sel aktif (misal A1, B5)
  • Worksheet Area — Grid baris (1,2,3...) dan kolom (A,B,C...) tempat data dimasukkan
  • Sheet Tab — Tab di bawah untuk berpindah antar sheet
  • Status Bar — Menampilkan info cepat (Sum, Average, Count) dari sel terpilih
Dasar

2. Tipe Data di Excel

Tipe DataContohCiri-ciri
Angka (Number)100, 3.14, -50Rata kanan, bisa dihitung
Teks (Text)"Halo", "A001"Rata kiri, tidak bisa dihitung
Tanggal (Date)26/05/2026Sebenarnya angka serial, bisa dihitung selisihnya
BooleanTRUE, FALSEHasil dari perbandingan logika
Error#VALUE!, #REF!, #N/AMenandakan ada kesalahan rumus

💡 Tips:

  • Angka yang diawali apostrof (') akan dianggap teks: '001 → teks "001"
  • Tanggal yang salah format bisa dianggap teks dan tidak bisa dihitung

4. Format Cell

Format Cell mengubah tampilan data tanpa mengubah nilainya. Akses via Ctrl+1.

FormatContoh InputTampilan
Number1234.51,234.50
Currency50000Rp50,000
Percentage0.8585%
Date4507326/05/2026

Dasar

5. Freeze Panes

Freeze Panes mengunci baris/kolom agar tetap terlihat saat scroll. Sangat berguna untuk tabel data yang panjang.

OpsiCaraFungsi
Freeze Top RowView → Freeze Panes → Freeze Top RowBaris 1 (header) tetap terlihat saat scroll ke bawah
Freeze First ColumnView → Freeze Panes → Freeze First ColumnKolom A tetap terlihat saat scroll ke kanan
Freeze CustomKlik sel → View → Freeze PanesMengunci baris di atas dan kolom di kiri sel aktif

💡 Tips:

  • Untuk freeze baris 1 dan kolom A sekaligus, klik sel B2 lalu Freeze Panes
  • Untuk membatalkan: View → Unfreeze Panes
