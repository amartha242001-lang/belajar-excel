⌨️ Shortcut Keyboard Excel

50+ shortcut keyboard untuk mempercepat pekerjaan Anda di Microsoft Excel.

📖 Cara Menggunakan Shortcut

Tekan tombol secara bersamaan. Contoh: Ctrl+C berarti tekan dan tahan tombol Ctrl, lalu tekan C, kemudian lepaskan keduanya.

🧭 Navigasi

Ctrl+Home
Pindah ke sel A1
Ctrl+End
Pindah ke sel terakhir yang berisi data
Ctrl+→
Pindah ke sel terisi berikutnya (kanan)
Ctrl+←
Pindah ke sel terisi berikutnya (kiri)
Ctrl+↑
Pindah ke sel terisi berikutnya (atas)
Ctrl+↓
Pindah ke sel terisi berikutnya (bawah)
Ctrl+G
Buka dialog Go To

🖱️ Seleksi

Ctrl+A
Pilih semua sel
Ctrl+Shift+End
Pilih dari sel aktif sampai sel terakhir
Ctrl+Space
Pilih seluruh kolom
Shift+Space
Pilih seluruh baris
Ctrl+Shift+→
Pilih sampai sel terisi terakhir (kanan)
Shift+Click
Pilih range dari sel aktif ke sel yang diklik

📋 Copy & Paste

Ctrl+C
Copy (salin)
Ctrl+X
Cut (potong)
Ctrl+V
Paste (tempel)
Ctrl+Alt+V
Paste Special (pilih format paste)
Ctrl+D
Fill Down (isi ke bawah)
Ctrl+R
Fill Right (isi ke kanan)

🎨 Format

Ctrl+B
Bold (tebal)
Ctrl+I
Italic (miring)
Ctrl+U
Underline (garis bawah)
Ctrl+1
Buka Format Cells dialog
Ctrl+Shift+!
Format angka dengan 2 desimal
Ctrl+Shift+$
Format mata uang
Ctrl+Shift+%
Format persen
Alt+H+B
Tambah border

📐 Rumus

Alt+=
AutoSum
F4
Toggle absolute/relative reference ($)
Ctrl+`
Tampilkan/sembunyikan rumus di sel
F2
Edit sel aktif (masuk mode edit)
Tab
Pilih saran autocomplete rumus
Ctrl+Shift+Enter
Masukkan rumus array (CSE)
F9
Hitung ulang semua workbook

💾 File

Ctrl+S
Simpan file
Ctrl+N
Buat workbook baru
Ctrl+O
Buka file
Ctrl+P
Print (cetak)
Ctrl+W
Tutup workbook aktif
F12
Save As (simpan sebagai)

📏 Baris & Kolom

Ctrl++
Sisipkan baris/kolom
Ctrl+-
Hapus baris/kolom
Alt+H+O+I
Auto-fit lebar kolom
Ctrl+9
Sembunyikan baris terpilih
Ctrl+Shift+9
Tampilkan baris tersembunyi
Ctrl+0
Sembunyikan kolom terpilih

📄 Sheet

Ctrl+Page Up
Pindah ke sheet sebelumnya
Ctrl+Page Down
Pindah ke sheet berikutnya
Shift+F11
Sisipkan sheet baru
Alt+H+D+S
Hapus sheet aktif
Alt+H+O+R
Rename sheet aktif

Find (cari)

Find & Replace
Ctrl+Shift+L
Toggle AutoFilter
Ctrl+T
Buat Table dari data
Ctrl+;
Masukkan tanggal hari ini
Ctrl+Shift+;
Masukkan waktu saat ini
Ctrl+E
Flash Fill
Alt+F1
Buat chart dari data terpilih
Esc
Batalkan input / tutup dialog
