📕 Level 4: Visualisasi & Pelaporan

Sajikan data dalam bentuk grafik dan laporan yang mudah dipahami.

Chart

1. Membuat Chart (Grafik)

Cara Cepat:
Pilih data → Insert → Chart (atau Alt+F1 untuk chart instan)

Jenis Chart & Kapan Digunakan:

Jenis ChartKapan DigunakanContoh
📊 Column/BarMembandingkan kategoriPenjualan per bulan
📈 LineMenunjukkan tren waktuPertumbuhan revenue
🥧 Pie/DonutMenunjukkan proporsiMarket share
📉 AreaTren dengan volumeTraffic website
⚡ ScatterKorelasi 2 variabelHarga vs Penjualan
🗺️ MapData geografisPenjualan per provinsi

💡 Tips:

  • Gunakan Recommended Charts jika bingung pilih jenis chart
  • Klik chart → Design tab → Change Chart Type untuk ganti jenis
  • Tambahkan Data Labels agar angka terlihat langsung di chart
Pivot

2. Pivot Table (Wajib Dikuasai!)

Cara Membuat:
Pilih data → Insert → PivotTable → Pilih lokasi → OK

4 Area Pivot Table:

AreaFungsiContoh
📋 RowsMengelompokkan data per barisNama Kota, Nama Produk
📊 ColumnsMengelompokkan data per kolomBulan, Kuartal
🔢 ValuesData yang dihitungSUM of Penjualan, COUNT of Order
🔎 FiltersMenyaring keseluruhan pivotFilter per Tahun

Contoh Kasus:

Data: 5000 baris transaksi dengan kolom Tanggal, Produk, Kota, Qty, Total

Pivot: Drag "Kota" ke Rows, "Produk" ke Columns, "Total" ke Values

Hasil: Tabel ringkasan total penjualan per kota per produk dalam hitungan detik!

💡 Tips:

  • Klik kanan pada tanggal → Group → pilih Months/Quarters untuk grouping otomatis
  • Gunakan Pivot Chart untuk visualisasi langsung dari Pivot Table
  • Refresh pivot (Alt+F5) setelah data sumber berubah
  • Gunakan Slicer (Insert → Slicer) untuk filter visual yang interaktif
Report

3. Conditional Formatting

Memberi warna/format otomatis berdasarkan nilai sel. Membuat data "berbicara" secara visual.

TipeFungsiContoh
Data BarsBar horizontal dalam selVisualisasi progress
Color ScalesGradasi warna (hijau-kuning-merah)Heatmap nilai
Icon SetsIkon (panah, lampu, bintang)Status KPI
Highlight RulesWarnai sel berdasarkan kondisiNilai > 90 = hijau
Report

4. Sparklines (Mini Chart dalam Sel)

Grafik kecil yang muat dalam satu sel. Cocok untuk menunjukkan tren tanpa membuat chart besar.

Cara:
Insert → Sparklines → Line/Column/Win-Loss → Pilih data range → OK
